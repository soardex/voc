voc is an OpenGL 3.2 and GLSL 1.5 modeled graphical rendering engine.

#### Dependecies:

+ [GLEW][0]
+ [Assimp][1]
+ [SDL 2][2]
+ [Bullet 3D Physics][3]
+ [Zlib][4]
+ [FreeImage][5]
+ [FreeType][6]
+ [PugiXML][7]
+ [Lua][8]
